# 🎬 Movies Sales Analysis Dashboard

A comprehensive **Movies Sales Analysis** project built using **Power BI** to analyze movie sales performance, revenue trends, customer insights, and business KPIs through interactive dashboards and visualizations. This project demonstrates the complete data analysis workflow, from data cleaning and exploration to dashboard creation and business insights.

---

## 📌 Project Overview

The objective of this project is to transform raw movie sales data into meaningful insights that help understand:

- 📈 Sales Performance
- 💰 Revenue Analysis
- 🎭 Genre-wise Performance
- 🌍 Regional Sales Trends
- 🏆 Top Performing Movies
- 📊 KPI Monitoring
- 📅 Time-based Sales Analysis

The dashboard enables decision-makers to quickly identify trends and make data-driven business decisions.
